# Project Idea: AI Personal Safety & Productivity Agent

This project is an AI-powered assistant designed especially for students and young women.

Main goals:
- Help with personal safety
- Improve daily productivity
- Provide mental and physical wellness guidance
- Work in low-internet conditions

The agent will work as:
1. A safety companion
2. A study helper
3. A confidence-building companion
4. A daily planner

This idea is inspired by real-life problems faced by students like me.
